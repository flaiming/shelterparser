Ignor

Plemeno: labrador
Pohlaví: pes
Věk: 1-2 roky
Datum předání do útulku: 22.2.2014...

Ignor je mladý, velmi temperamentní a dominantí pes, který zjevně svým původním majitelům přerostl přes hlavu. K lidem i dětem je přátelský, rád se mazlí a potřebuje aktivní páníčky, protože se jen tak neunaví. Zvládá základní povely, ale rychle by se učil i další. Bohužel se velmi špatně snáší s jinými psy, útočí na ně a je agresivní.
